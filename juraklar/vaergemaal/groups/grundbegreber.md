# Gruppe: Grundbegreber

Denne gruppe samler **definitioner og forklaringer** af centrale juridiske begreber inden for értgemål.

## Primært dokument

[Juridiske grundbegreber — értgemål](../notes/juridiske-grundbegreber-vaergemaal.md)

---

## Hurtig begrebsoversigt

| Begreb | Kort definition |
|---|---|
| Értgemål | Retslig beskyttelsesordning for voksne der ikke kan varetage egne interesser |
| Érte | Person beskikket af Familieretshuset til at handle på vegne af den berørte |
| Handleevne | Evnen til at foretage retsgyldige handlinger (kontrakter, dispositioner) |
| Fuld umyndiggørelse | Tab af handleevne fuldstændigt (§ 6) — meget sjælden |
| Partshøring | Lovpligtig mulighed for den berørte at udtale sig inden afgørelse |
| Bisidder | Støtteperson uden selvstændig juridisk mandat |
| Fremtidsfuldmagt | Frivillig fuldmagt oprettet i tide — foretrukket alternativ til értgemål |
| Mindsteindgrebsprincippet | Den mindst indgribende foranstaltning skal altid vælges |
| Familieretshuset | Administrativ myndighed der behandler értgemålssager |
| Familieretten | Byrettens familieafdeling — domstolsbehandling ved tvist |
| Årsregnskab | Løbende regnskab som économisk érte aflægger til Familieretshuset |
| Dispositionsgrænse | Beløbsgrænse (ca. 30.000 kr.) over hvilken Familieretshuset godkendelse kræves |
| Aktindsigt | Ret til at se alle dokumenter i sin sag |
| Beskikkelse | Formel udnævnelse af érte af Familieretshuset |
| Personlige anliggender | Sundhed, bolig, daglig livsførelse |
| Ekonomiske anliggender | Bank, formue, kontrakter, gæld |
