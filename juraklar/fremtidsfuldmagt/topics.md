# Nøgletermer — Fremtidsfuldmagt

Normaliserede nøgletermer og synonymer til brug for søgning og indeksering.

---

## Juridiske termer

| Normaliseret term | Synonymer og varianter |
|---|---|
| fremtidsfuldmagt | fremtidig fuldmagt, living will (forkert brug), selvbestemmelsesdokument |
| fuldmagtsgiver | den der opretter fuldmagten, ophavsmand til fuldmagt |
| fuldmagtshaver | befuldmægtiget, repræsentant, administrator |
| habilitet | handleevne, retslig handledygtighed, mental kapacitet |
| inhabilitet | manglende handleevne, kognitiv svækkelse |
| notarvedståelse | vedståelse for notar, notarisering, notarbekræftelse |
| aktivering | ikrafttrædelse af fuldmagt, iværksættelse |
| tilbagekaldelse | ophævelse, annullering af fuldmagt |
| lægeerklæring | helbredserklæring, habituerklæring |
| værgemål | kuratel, personlig hjælp (forældet), statslig formynderskab |
| Familieretshuset | statsforvaltningen (ældre term), familieretslig myndighed |
| Tinglysningsretten | tinglysning, det digitale tinglysningssystem |
| Fremtidsfuldmagtsregistret | fuldmagtsregistret, CPR-koblet register |

---

## Processuelle termer

| Normaliseret term | Synonymer og varianter |
|---|---|
| digital oprettelse | onlineoprettelse, oprettelse via tinglysning.dk |
| notarvedståelse | møde hos notaren, notarbevilling |
| aktiveringsansøgning | ansøgning om aktivering, anmodning til Familieretshuset |
| sagsbehandling | behandling af ansøgning, myndighedsbehandling |
| tilsyn | kontrol, overvågning, regnskabsgennemgang |
| bekymringsindberetning | indberetning, anmeldelse til Familieretshuset |
| klage | klageret, anke, indbringelse for Familieretten |

---

## Relevante lovgivningsreferencer

- Lov om fremtidsfuldmagter: lovnr. 618, L618, lov 618/2016
- Bekendtgørelse om fremtidsfuldmagter: bkg. 1018, B1018, bekendtgørelse 1018/2017
- Myndighedsloven: LBK nr. 1015/2007
- Straffeloven § 280: misbrug af særlig tillid
- GDPR: databeskyttelse i Fremtidsfuldmagtsregistret

---

## Målgrupper

- Ældre borgere: seniorer, pensionister, ældre, 60+, 70+
- Pårørende: familie, børn, ægtefælle, partner, søskende
- Fuldmagtshavere: befuldmægtigede, udpegede repræsentanter
- Professionelle: advokater, revisorer, bankrådgivere, plejepersonale, læger

---

## Relaterede emner

- Testamente: arv, skiftebehandling, arveret
- Ægtepagt: formueforhold, særeje, fælleseje
- Patientrettigheder: behandlingstestamente, livstestamente
- Selskabsret: prokura, tegningsret, successionplanlægning
- Demenssygdom: Alzheimers, Parkinsons, kognitiv svækkelse
