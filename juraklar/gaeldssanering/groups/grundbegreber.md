## Vigtigste Begreber — Hurtigt Overblik

### Centrale Juridiske Begreber

| Begreb | Kort forklaring |
|--------|----------------|
| Akkord | Aftale om delvis gieldseftergivelse |
| Dividende | Procent af krav som kreditorer modtager |
| Håbløs gæld | Gæld der aldrig realistisk kan betales |
| Insolvens | Passiver overstiger aktiver |
| Kendelse | Retsafgørelse fra Skifteretten |
| Panteret | Sikkerhedsret i aktiv |
| Pro-rata | Forholdsmæssig fordeling |
| Rådighedsbeløb | Hvad skyldner beholder til underhold |
| Saneringsforslag | Konkrete vilkår for gældssaneringen |
| Skifteretten | Retten der behandler gældssaneringssager |

### Kreditortyper

| Type | Beskrivelse | Omfattet af sanering? |
|------|------------|----------------------|
| Sikrede kreditorer | Har pant (realkreditinstitut, panthavere) | NEJ |
| Usikrede kreditorer | Intet pant (forbrugslångivere, SKAT mv.) | JA |

### Gieldstyper og Sanerbarhed

| Gieldstype | Kan saneres? |
|-----------|-------------|
| Forbrugslån | Ja |
| Banklån (usikret) | Ja |
| Skattegæld | Ja |
| Studielån (SU) | Som udgangspunkt ikke |
| Realkreditlån | Nej — pantesikret |
| Bøder fra det offentlige | Nej |
| Erstatning fra kriminalitet | Nej |
