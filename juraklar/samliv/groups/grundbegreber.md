# Gruppe: Grundbegreber

Denne gruppe samler begrebsforklarende materiale — juridiske termer og begreber der går igen på tværs af emnerne i Samlivsaftale (Ugifte)-wikien.

## Dokumenter i Gruppen

- [Juridiske grundbegreber — samliv](../notes/juridiske-grundbegreber-samliv.md)
  Alfabetisk ordliste med forklaringer på centrale juridiske begreber i konteksten af ugiftes samlivsret.

## Centrale Begreber — Hurtig Reference

| Begreb | Kort forklaring |
|---|---|
| Samlivsaftale | Privatretlig aftale om fælles økonomi og praktik |
| Samejeoverenskomst | Aftale om fælles ejerskab af fast ejendom; tinglyses |
| Samlevertestamente | Testamente efter arvelovens § 87; giver 15% boafgift |
| Tvangsarv | 25% af bo der ubetinget tilkommer livsarvinger |
| Friarv | Den del der frit kan testamenteres (op til 75%) |
| Boafgift | Skat af arv; 15% for nære, 36,25% for andre |
| Begunstiget | Navngiven modtager af pensionsudbetaling |
| Forkøbsret | Ret til at overtage andens ejerandel ved ophør |
| Tinglysning | Offentlig registrering af ejendomsrettigheder |
| Solidarisk hæftelse | Begge hæfter for hele den fælles gæld |
| CRT | Centralregistret for Testamenter |
| Sameje | Fælles ejerskab, formodning om 50/50 uden aftale |

## Brug af Begrebsnotatet

Begrebsnotatet er tænkt som et opslagsværk til brug for:
- Rådgivere der vil forklare et begreb for en klient
- Brugere der læser artiklerne og støder på ukendte termer
- Sammenstilling af begreber på tværs af temaer

Alle begreber linker til relevant uddybning i artiklerne.
