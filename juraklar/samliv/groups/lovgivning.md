# Gruppe: Lovgivning

Denne gruppe samler alle dokumenter der direkte behandler lovgrundlaget for samboende i Danmark.

## Dokumenter i Gruppen

### Politikker (Lovanalyser)

- [Arveloven og samlevende](../policies/arveloven-og-samlevende.md)
  Gennemgang af arvelovens regler for samboende, herunder § 87-ordningen, boafgiftssatser og testamentekrav.

- [Aftaleloven og samlivsaftale](../policies/aftaleloven-og-samlivsaftale.md)
  Aftalelovens rolle som hjemmel for samlivsaftaler. Gyldighedsbetingelser, § 36-generalklausulen og bevisregler.

### Rå Retskilder

- [Ugiftes retslige stilling](../raw/ugiftes-retslige-stilling.md)
  Samlet oversigt over ugiftes manglende rettigheder vs. ægtefæller.

- [Samlivsaftalen og dens indhold](../raw/samlivsaftalen-og-indhold.md)
  Hvad aftalen kan og ikke kan regulere.

- [Samejeoverenskomst](../raw/samejeoverenskomst.md)
  Retsgrundlaget for samejeoverenskomsten og tinglysningskrav.

- [Arv og testamente for samlevende](../raw/arv-og-testamente-for-samlevende.md)
  Arvelovens regler, § 87, boafgift og testamentekrav.

- [Pension og forsikring samlevende](../raw/pension-og-forsikring-samlevende.md)
  Regler for begunstigede og pensionsudbetalinger.

| Tinglysningsloven | Samejeoverenskomstens tinglysning |
| Lejeloven | Rettigheder ved fælles lejebolig |
| Boafgiftsloven | Boafgiftssatser ved arv |
