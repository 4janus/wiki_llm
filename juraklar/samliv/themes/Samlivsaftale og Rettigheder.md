# Tema: Samlivsaftale og Rettigheder

Dette tema samler alt materiale der vedrører samlivsaftalens indhold, ugifte pars retlige stilling og den generelle retslige ramme for samboende i Danmark.

## Temaets Centrale Spørgsmål

- Hvad kan og bør en samlivsaftale indeholde?
- Hvilke rettigheder har ugifte samboende — og hvilke mangler de?
- Hvad er forskellen på at bo sammen og at være gift?
- Kan vi lave en samlivsaftale selv, eller behøver vi advokat?
- Hvad er den juridiske status for vores aftale?

## Retsligt Udgangspunkt

Ugifte samboende i Danmark er retligt set fremmede for hinanden. Der eksisterer ingen samlivslov der automatisk giver samboende rettigheder svarende til ægtefæller. Samlivsaftalen er den privatretlige mekanisme der kan udfylde dette tomrum — inden for de grænser aftaleloven sætter.

De vigtigste manglende rettigheder for ugifte:
- Ingen automatisk arveret
- Ingen bodeling ved ophør
- Ingen ret til ægtefællebidrag
- Ingen ret til fælles bolig ved ophør
- Ingen automatisk ret til partners pension

## Hvad Samlivsaftalen Kan Regulere

En velskrevet samlivsaftale kan adressere:
- Løbende udgiftsfordeling (husleje, forsyninger, dagligvarer)
- Ejerforhold til indbo, løsøre og fælles anskaffelser
- Lejebolig — hvem bevarer retten ved ophør
- Kompensation hvis den ene sætter karriere til side
- Fælles gæld og intern hæftelsesfordeling

Samlivsaftalen kan ikke erstatte testamente, samejeoverenskomst eller pensionsbegunstigelse — disse kræver separate tiltag.

## Nøglepunkter for Rådgivning

1. Start med at kortlægge hvad parret ønsker at regulere
2. Skeln mellem det aftalen kan regulere og det der kræver andre dokumenter
3. Brug aftaleloven som hjemmel — ingen særlig samlivslov
4. Vær opmærksom på § 36-generalklausulen ved skæve magtforhold
5. Opdater aftalen ved store livsbegivenheder
