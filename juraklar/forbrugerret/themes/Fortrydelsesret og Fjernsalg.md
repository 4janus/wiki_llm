# Tema: Fortrydelsesret og Fjernsalg

Dette tema dækker reglerne om fortrydelsesret ved køb via internet, telefon og andre fjernkanaler.

## Hvad dækker dette tema?

- Hvornår fortrydelsesretten gælder (og ikke gælder)
- Fristen på 14 kalenderdage
- Sådan fortryder man korrekt
- Returnering og tilbagebetaling
- Sælgers informationspligt
- Fjernsalgsreglernes særlige beskyttelse

## Nøglepunkter

**Fortrydelsesretten er kanalbundet:** Retten gælder ved fjernsalg (netbutik, telefon, katalog) og dørsalg — IKKE ved køb i fysisk butik. Dette er den hyppigste misforståelse i forbrugerret.

**14 dage fra levering:** Fristen løber fra den dag varen modtages — ikke fra bestillingsdagen.

**Ingen begrundelse nødvendig:** Du kan fortryde uden at give en begrundelse.

**Sælger kan begrænse tilbagebetalingen:** Har du taget varen i egentlig brug (ikke kun undersøgt den), kan sælger reducere tilbagebetalingen svarende til værdiformindskelsen.

**Returomkostninger:** Som udgangspunkt betaler du returfragt, medmindre sælger har lovet gratis returnering.

**Manglende information forlænger fristen:** Oplyser sælger ikke om fortrydelsesretten, forlænges fristen med 12 måneder.

## Undtagelser — hvornår gælder fortrydelsesretten ikke?

| Situation | Fortrydelsesret? |
|---|---|
| Netbutik | Ja |
| Fysisk butik | Nej |
| Telefonsalg | Ja |
| Specialfremstillet vare | Nej |
| Åbnet hygieneprodukt | Nej |
| Downloadet digitalt indhold | Nej |
| Flybilletter med fast dato | Nej |
