# Tema: Huslejenævnet og Tvister

Dette tema samler alt om tvistløsning i lejeret — fra Huslejenævnet til Boligretten.

---

## Hvad dækker dette tema?

Temaet dækker:

- Huslejenævnets rolle, kompetence og sagsbehandling
- Boligrettens rolle og kompetence
- Klageprocesser og frister
- Huslejeregulering
- Praktiske råd ved tvister

## Nøglefakta om Huslejenævnet

| Parameter | Detalje |
|---|---|
| Type | Kommunalt klagenævn |
| Gebyr (2024) | Ca. 166 kr. |
| Behandlingstid | 3-12 måneder |
| Advokatpligt | Nej |
| Anke til Boligretten | Inden 4 uger fra afgørelse |

---

## Hvad kan Huslejenævnet behandle?

- For høj husleje (lejeregulerede kommuner)
- Fraflytningskrav
- Depositumreturnering
- Vedligeholdelsespligter
- Varmeregnskab
- Husdyr
- Fremleje
- Udlejers ulovlige adgang
- Ugyldige kontraktvilkår

---

## Hvad kan Huslejenævnet IKKE behandle?

- Opsigelse og ophævelse (går til Boligretten)
- Fysisk udsættelse (fogedretten)
