# Gruppe: Grundbegreber

Denne gruppe samler definitioner og forklaringer på de grundlæggende juridiske begreber i lejeret.

---

## Noter

- [Juridiske grundbegreber i lejeret](../notes/juridiske-grundbegreber-lejeret.md) — alfabetisk opslagsværk over lejeretlige termer

---

## Vigtige grundbegreber — oversigt

| Begreb | Kort forklaring |
|---|---|
| Depositum | Sikkerhedsbeløb maks. 3 måneders nettoleje |
| Forudbetalt leje | Forudbetaling maks. 3 måneder |
| Nettoleje | Leje ekskl. aconto-bidrag |
| Normal slid | Naturlig forringelse — udlejer betaler |
| Misligholdelse | Skader udover normal brug — lejer betaler |
| Indflytningsrapport | Dokumentation af stand ved indflytning |
| Fraflytningsrapport | Dokumentation af stand ved fraflytning |
| Det lejedes værdi | Markedsniveau for sammenlignelige lejemål |
| Præceptiv regel | Kan ikke fraviges til skade for lejer |
| Huslejenævnet | Kommunalt klagenævn for lejetvister |
| Boligretten | Byretten for lejeretlige ankesager |

---

## Nøglefrister i lejeret

| Frist | Hvad |
|---|---|
| 2 uger fra indflytning | Lejer anmelder mangler skriftligt |
| 6 uger fra fraflytning | Udlejer skal fremsende krav |
| 6 uger fra modtagelse | Lejer gør indsigelse mod krav |
| 2 uger (privat udlejer) | Depositum returneres |
| 4 uger (selskabsudlejer) | Depositum returneres |
| 4 måneder | Udlejer sender varmeregnskab |
| 6 uger fra varmeregnskab | Lejer gør indsigelse |
| 4 uger fra afgørelse | Anke til Boligretten |
