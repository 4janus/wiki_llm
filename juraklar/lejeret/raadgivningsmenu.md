## Nøglefrister — husk disse

| Frist | Handling |
|---|---|
| **2 uger** fra indflytning | Anmeld mangler skriftligt til udlejer |
| **6 uger** fra fraflytning | Udlejers krav skal fremsendes |
| **6 uger** fra modtagelse | Din indsigelse mod udlejers krav |
| **2 uger** (privat udlejer) | Depositum returneres |
| **4 uger** (selskabsudlejer) | Depositum returneres |
| **4 måneder** efter regnskabsår | Varmeregnskab modtages |
| **6 uger** fra varmeregnskab | Din indsigelse mod varmeregnskabet |
| **4 uger** fra nævnsafgørelse | Anke til Boligretten |
