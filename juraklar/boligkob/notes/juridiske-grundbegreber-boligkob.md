# Juridiske Grundbegreber — Boligkøb

## Resumé

Boligkøb i Danmark involverer en lang række juridiske begreber, der er centrale for at forstå processen og dine rettigheder. Nedenstående ordliste forklarer de vigtigste termer i forbindelse med fast ejendom og bolighandel på dansk.

## Vigtige Emner

- **Tinglysning**: officiel registrering af ejendomsrettigheder
- **Skøde**: overdragelsesdokument der overfører ejerskab
- **Servitut**: tinglyst byrde der begrænser ejendomsretten
- **Advokatforbehold**: købers ret til juridisk gennemgang inden for 6 hverdage
- **Refusionsopgørelse**: fordeling af løbende udgifter ved overtagelse
- **Ejerskifteforsikring**: forsikring mod skjulte fejl og mangler

## Detaljer

### A. Ejendomsretlige Grundbegreber

| Begreb | Definition |
|---|---|
| Fast ejendom | Jord og de bygninger der er permanent befæstet til jorden |
| Matrikelnummer | Ejendommes officielle identifikationsnummer i matriklen |
| Tingbog | Offentligt register over rettigheder over fast ejendom |
| Tingbogsattest | Officielt udtræk fra Tingbogen med alle rettigheder og byrder |
| Skøde | Overdragelsesdokument der formelt overfører ejerskab fra sælger til køber |
| Tinglysning | Registrering af rettigheder i Tingbogen, der giver retsvirkning over for tredjemand |
| Prioritet | Rækkefølgen af tinglyste rettigheder (vigtig ved tvangsauktion) |

### B. Finansieringsrelaterede Begreber

| Begreb | Definition |
|---|---|
| Realkreditlån | Lån med pant i fast ejendom, ydet af realkreditinstitut — op til 80% af ejendommens værdi |
| Pantebrev | Dokument der stifter pant i fast ejendom som sikkerhed for et lån |
| Hæftelse | Pant eller anden byrde der hviler på ejendommen og er registreret i Tingbogen |
| Bidragssats | Løbende gebyr til realkreditinstituttet oven i renten |
| Belåningsgrad | Lånets størrelse i procent af ejendommens vurderingsværdi |
| Kursskæring | Forskel mellem kursværdi og nominel lånestørrelse ved realkreditlån |

### C. Juridiske Processer og Dokumenter

| Begreb | Definition |
|---|---|
| Advokatforbehold | Klausul i købsaftale der giver køber 6 hverdage til advokatgennemgang |
| Købsaftale | Juridisk bindende aftale om køb af fast ejendom |
| Refusionsopgørelse | Forholdsmæssig fordeling af løbende udgifter (skat, vand mv.) fra overtagelsesdagen |
| BBR | Bygnings- og Boligregistret — officielle register over alle bygninger og boliger |
| Energimærke | Vurdering af ejendommens energiforbrug — obligatorisk ved salg |
| Lokalplan | Kommunens detaljerede plangrundlag for et geografisk område |

### D. Huseftersynsordningens Begreber

| Begreb | Definition |
|---|---|
| Tilstandsrapport | Beskikket bygningssagkyndigs gennemgang og vurdering af ejendommens fysiske tilstand |
| Elinstallationsrapport | Autoriseret el-installatørs gennemgang af de elektriske installationer |
| K0-K3 skadesklasser | Klassificering af skader fra kosmetiske (K0) til alvorlige (K3) |
| Ejerskifteforsikring | Forsikring der dækker skjulte fejl som ikke fremgik af rapporterne |
| Beskikket bygningssagkyndig | Autoriseret fagperson godkendt til at udarbejde tilstandsrapporter |
| Huseftersynsnævnet | Klageorgan for tilstandsrapporter |

### E. Servitutter og Byrder

| Begreb | Definition |
|---|---|
| Servitut | Tinglyst ret eller begrænsning der hviler på ejendommen og følger ejendommen ved salg |
| Byggelinjeservitut | Fastlægger, at bygninger ikke må opføres tættere på vej eller skel end en bestemt linje |
| Brugsretsservitut | Giver en navngiven part ret til at bruge en del af ejendommen (fx sti, parkeringsplads) |
| Rørservitut | Ret for forsyningsselskab til at have ledninger gennem grunden |
| Fredning | Offentligretlig begrænsning af anvendelse og bebyggelse (kulturarv eller natur) |
| Deklaration | Erklæring tinglyst på ejendommen, der fastlægger rettigheder og forpligtelser |

### F. Boligskatter

| Begreb | Definition |
|---|---|
| Ejendomsværdiskat | Statslig skat beregnet som procent af ejendommens offentlige vurdering |
| Grundskyld | Kommunal skat af grundens (jordens) offentlige vurdering |
| Ejendomsvurdering | Vurderingsstyrelsens offentlige vurdering af ejendommens markedsværdi |
| Grundværdi | Offentlig vurdering af grundens værdi uden bygninger |
| Boligskatteloft | Overgangsordning der begrænser max. stigning i boligskat per år |

### G. Ejerlejlighedsspecifikke Begreber

| Begreb | Definition |
|---|---|
| Ejerforening | Forening af alle lejlighedsejere i en ejendom — administrerer fællesarealerne |
| Fællesudgifter | Månedlige bidrag til ejerforeningen til drift og vedligehold af fællesarealer |
| Fordelingstal | Ejerandelen af fællesejendommen — bruges til at beregne fællesudgifter og stemmevægt |
| Ejerlejlighedsplan | Tegning der afgrænser den individuelle lejligheds grænser i ejendommen |
| Fællesgæld | Lån optaget af ejerforeningen til fælles renoveringer — fordelt på ejerne |
| Generalforsamling | Ejerforeningens øverste beslutningsorgan — mødes mindst én gang om året |
