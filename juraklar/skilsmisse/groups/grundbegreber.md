# Gruppe: Grundbegreber

Denne gruppe samler ordbøger, begrebsforklaringer og terminologi-ressourcer for separation og skilsmisse i Danmark.

## Centrale Begreber — Hurtigoversigt

| Begreb | Kort forklaring |
|--------|----------------|
| Separation | Juridisk adskillelse — ægteskab ophæves ikke |
| Skilsmisse | Endelig ophævelse af ægteskabet |
| Familieretshuset | Administrativ myndighed for familieretlige sager |
| Familieretten | Byretsafdeling for tvistede familiesager |
| Delingsformue | Fælles formue der deles 50/50 |
| Særeje | Formue der holdes udenfor bodelingen |
| Skæringsdato | Dato for formueopgørelse ved bodeling |
| Ægtefællebidrag | Løbende ydelse til svagere ægtefælle, max 10 år |
| Normalbidrag | Standardbørnebidrag ca. 1.682 kr./md. (2024) |
| Grønt/gult/rødt spor | Familieretshusets tre sagsbehandlingsspor |

## Anvendelse

Grundbegrebsgruppen er det rette udgangspunkt for dig, der er ny i systemet og ønsker at forstå de termer, du møder i breve fra Familieretshuset, i advokatkommunikation eller i artikler om emnet.
