# Tema: Bodeling og Formue

## Temaets Fokus

Dette tema dækker den formelle fordeling af ægtefællernes fælles formue ved skilsmisse — hvad der indgår, hvad der holdes udenfor, og hvordan processen for bodeling forløber. Bodelingen er et separat processuel spor fra selve skilsmissen og kan gennemføres enten ved privat aftale eller via skifteretten.

## Centrale Spørgsmål

- Hvad er delingsformuen, og hvad er særeje?
- Hvad er bodelingsskæringsdatoen?
- Hvad sker der med pensioner ved skilsmisse?
- Hvad er en ægtepagt, og er det for sent at oprette en?
- Hvordan behandles fælles gæld?
- Hvad sker der med virksomhed og investeringer?
- Hvad sker der, hvis vi ikke kan aftale en bodeling?

## Nøgleregler

**Udgangspunkt:** Alt deles 50/50 (nettoværdi efter gæld).

**Undtagelse særeje:** Kræver ægtepagt eller arv/gave med særejevilkår.

**Pensioner:** Rimelige pensioner holdes normalt udenfor bodeling. Pensionskompensation kan tilkendes.

**Skæringsdato:** Normalt separationsdatoen — alt efter denne dato er privat.

**Skifteretten:** Kan behandle tvistede bodelinger og udpege bobestyrer.

## Formuetyper ved bodeling

| Formue-type | Hvad er det? | Deles? |
|-------------|-------------|--------|
| Delingsformue | Alt ægtefæller ejer på skæringsdatoen | Ja, 50/50 |
| Særeje (aftalt) | Fastsat ved ægtepagt | Nej |
| Arvesæreje | Arv/gave med særejevilkår | Nej |
| Rimelige pensioner | Arbejdsmarkedspension mv. | Normalt nej |
| Aktiver efter skæringsdato | Erhvervet efter adskillelse | Nej |

## Bodelingens procesmuligheder

| Metode | Beskrivelse | Varighed |
|--------|-------------|---------|
| Privat aftale | Parterne aftaler selv, evt. med advokat | Fleksibel |
| Skifteretsbehandling | Tvangsdeling via domstol | Måneder til år |
| Bobestyrer | Udpeget af skifteretten til at lede delingen | Variabel |
