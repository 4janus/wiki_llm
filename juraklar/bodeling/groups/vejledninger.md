## Processen kort fortalt

```
1. Ansøg om separation → Familieretshuset
2. Fastlæg skæringsdato
3. Fremlæg kapitaloplysninger (begge parter)
4. Dokumenter særeje
5. Beregn nettoformuer
6. Forhandl og aftal fordeling
7. Underskriv bodelingsaftale
8. Tinglys ejendomsoverdragelse (hvis relevant)
9. Gennemfør udligningsbetaling
```

## Checkliste til bodeling

- [ ] Skæringsdato fastlagt og dokumenteret
- [ ] Alle bankkonti opgjort pr. skæringsdato
- [ ] Ejendom vurderet (mæglervurdering)
- [ ] Pensioner opgjort via Pensionsinfo.dk
- [ ] Virksomhed vurderet (revisor)
- [ ] Bil vurderet til markedsværdi
- [ ] Særeje dokumenteret (ægtepagt/gavebrev/testamente)
- [ ] Nettoformuer beregnet for begge parter
- [ ] Udligningsbetaling beregnet
- [ ] Bodelingsaftale underskrevet
- [ ] Ejendomsoverdragelse tinglyst (hvis relevant)

## Udgifter at forvente

| Post | Estimeret pris |
|------|---------------|
| Advokat (ukompliceret) | 10.000–40.000 kr. |
| Mæglervurdering (ejendom) | 5.000–10.000 kr. |
| Revisorvurdering (virksomhed) | 15.000–50.000 kr. |
| Tinglysning (ejendom) | 1.850 kr. + 0,6 % |
| Skifteretssag (ved uenighed) | 50.000–150.000+ kr. |
