## Nøgleprincipper

| Princip | Forklaring |
|---------|-----------|
| Ligedeling | De samlede positive nettoformuer deles 50/50 |
| Skæringsdato | Tidspunktet for formuefrysningen (normalt dato for separationsansøgning) |
| Nettoformue | Aktiver minus passiver pr. skæringsdato |
| Delingsformue | Al formue der ikke er særeje, rimelig pension eller personlig erstatning |
| Negativ nettoformue | Bidrager ikke til bodelingen — kun positive tæller |

## Hurtig reference — udgifter

| Post | Estimeret pris |
|------|---------------|
| Advokat (ukompliceret bodeling) | 10.000–40.000 kr. |
| Tinglysning ved ejendomsoverdragelse | 1.850 kr. + 0,6 % |
| Ejendomsvurdering (privat mægler) | 5.000–10.000 kr. |
| Virksomhedsvurdering | 15.000–50.000 kr. |
| Skifteretssag | 50.000–150.000+ kr. |
