# Juridiske grundbegreber — Bodeling

## Kernebegreber

| Begreb | Definition |
|--------|-----------|
| **Bodeling** | Den juridiske proces hvorved ægtefællers formue fordeles ved ægteskabets ophør (separation, skilsmisse eller dødsfald) |
| **Delingsformue** | Den del af en ægtefælles formue der skal ligedeles med den anden ægtefælle ved bodeling |
| **Særeje** | Formue der holdes udenfor bodeling — enten via ægtepagt eller tredjemandsvilkår |
| **Nettoformue** | En ægtefælles samlede aktiver minus samlede passiver pr. skæringsdatoen |
| **Skæringsdato** | Det tidspunkt hvorfra formuen opgøres — normalt datoen for ansøgning om separation/skilsmisse |
| **Ligedelingsprincippet** | Udgangspunktet i dansk familieret: de samlede positive nettoformuer deles 50/50 |
| **Bodelingsbalance** | Det regnskab der opstilles over begge ægtefællers aktiver og passiver som grundlag for bodelingen |
| **Bodelingsaftale** | Den privatretlige kontrakt hvori parterne dokumenterer og aftaler fordelingen af formue |

## Aktiver og passiver

| Begreb | Definition |
|--------|-----------|
| **Aktiver** | Alt hvad en ægtefælle ejer af positiv værdi: ejendom, bil, bankkonti, aktier mv. |
| **Passiver** | Al gæld og forpligtelser: realkreditlån, banklån, forbrugslån, skattegæld |
| **Handelspris** | Den pris et aktiv ville opnå ved normalt salg på det åbne marked |
| **Indre værdi** | Bogført egenkapital i et selskab — udgangspunkt for vurdering af unoterede aktier |
| **Goodwill** | Immateriel værdi af en virksomhed ud over de fysiske aktiver |
| **Andelsværdi** | Den af andelsboligforeningen fastsatte maksimalpris for en andelsbolig |

## Særeje-begreber

| Begreb | Definition |
|--------|-----------|
| **Fuldstændigt særeje** | Særeje der beskytter aktivet mod bodeling OG mod den anden ægtefælles kreditorer |
| **Skilsmissesæreje** | Særeje der kun beskytter ved bodeling (skilsmisse), ikke ved dødsfald eller over for kreditorer |
| **Ægtepagt** | Skriftlig, tinglyst aftale mellem ægtefæller om formueordning — grundlag for særeje |
| **Tredjemandsvilkår** | Bestemmelse fra arvelader (testamente) eller gavegiver (gavebrev) om at arv/gave er modtagerens særeje |
| **Tinglysning** | Officiel registrering af rettigheder i Tinglysningsregistret — krav for gyldighed af ægtepagt |
| **Personbogen** | Det register hvori ægtepagter tinglyses |

## Procesuelle begreber

| Begreb | Definition |
|--------|-----------|
| **Separation** | Juridisk adskillelse af ægtefæller — forudsætning for bodeling, men ægteskabet fortsætter |
| **Skilsmisse** | Ophævelse af ægteskabet — bodeling gennemføres typisk i forbindelse hermed |
| **Familieretshuset** | Myndighed der behandler ansøgninger om separation og skilsmisse i Danmark |
| **Skifteretten** | Domstolsafdeling der behandler uenige bodelingssager |
| **Bodelingsbehandling** | Skifterettens behandling af en bodeling når parterne ikke kan blive enige |
| **Skønsmand** | Uafhængig ekspert udpeget af retten til at vurdere aktiver ved tvist |
| **Minskiftesag.dk** | Digital portal til ansøgning om bodelingsbehandling ved Skifteretten |

## Krav og kompensation

| Begreb | Definition |
|--------|-----------|
| **Kompensationskrav** | Krav efter Ægtefælleloven § 87 om kompensation for urimeligt formindsket delingsformue |
| **Udligningsbetaling** | Den betaling den rigere ægtefælle foretager til den fattigere for at udligne formueforskellen |
| **Naturaludlæg** | Overtagelse af et aktiv (f.eks. ejendommen) som led i bodelingen frem for kontant betaling |
| **Suppleringsarv** | Krav fra arvinger om at overlevende ægtefælle supplerer arven op — relevant ved dødsfald |

## Pensionsbegreber

| Begreb | Definition |
|--------|-----------|
| **Rimelig pension** | Pensionsordning der er forholdsmæssig med ægteskabets varighed og parternes alder — holdes udenfor bodeling |
| **Urimelig pension** | Pension der overstiger det rimelige — kan udløse kompensationskrav |
| **Pensionsoverførsel** | Direkte overførsel af pensionsmidler fra den ene ægtefælle til den anden — alternativ til kompensation |
| **Pensionsinfo.dk** | Offentlig platform der samler alle pensionsoplysninger med MitID-login |
| **ATP** | Arbejdsmarkedets Tillægspension — holdes normalt udenfor bodeling |

## Ejendomsbegreber

| Begreb | Definition |
|--------|-----------|
| **Skøde** | Dokument der overdrager ejendomsret til fast ejendom — skal tinglyses |
| **Tinglysningsafgift** | Afgift ved tinglysning af skøde: 1.850 kr. + 0,6 % af overdragelsessummen |
| **Realkredit** | Lån med pant i fast ejendom — restgæld modregnes i ejendomsværdien |
| **Vurderingsstyrelsen** | Statslig myndighed der foretager offentlige ejendomsvurderinger |

## Arveretlige begreber

| Begreb | Definition |
|--------|-----------|
| **Uskiftet bo** | Ordning hvorved den overlevende ægtefælle overtager hele fællesboet og beholder det uden at dele |
| **Livsarving** | Afdødes direkte efterkommer (barn, barnebarn) — har ret til tvangsarv |
| **Tvangsarv** | Den del af arven som livsarvinger har ubetinget ret til — kan ikke testamenteres bort |
| **Bobestyrer** | Advokat udpeget af retten til at lede behandlingen af et dødsbo |
| **Privat skifte** | Arvingerne behandler selv dødsboet (typisk med advokatbistand) |
