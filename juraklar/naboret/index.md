## Instansoversigt

| Problem | Klageinstans |
|---------|--------------|
| Hegn og skel | Hegnsynet |
| Nabotræer | Hegnsynet |
| Støj, røg, lugt | Kommunens tekniske forvaltning |
| Ulovligt byggeri | Kommunens byggemyndighed |
| Byggetilladelse | Planklagenævnet (inden 4 uger) |
| Lokalplansbrud | Kommunen → Planklagenævnet |
| Servitutbrud | Domstolene |
| Skelstrid | Landinspektør → Hegnsynet → Domstolene |
