## Vigtigste begreber — hurtigoversigt

| Begreb | Kort definition |
|--------|----------------|
| Immission | Indvirkning fra én ejendom på en anden (støj, røg, lugt, lys) |
| Tålelighedsniveauet | Den gene man normalt må acceptere i det pågældende område |
| Hegn | Afspærring langs ejendomsskel; reguleres af Hegnsloven |
| Skel | Juridisk grænse mellem to ejendomme; fremgår af matrikelkortet |
| Matrikelkortet | Offentlig kortværk over alle ejendomme og skel |
| Landinspektør | Statsautoriseret fagperson til matrikulær opmåling |
| Hegnsynet | Kommunalt nævn der afgør hegn- og skeltvister |
| Servitut | Tinglyst begrænsning eller rettighed på ejendom |
| Tingbogen | Offentligt register over rettigheder på fast ejendom |
| Lokalplan | Kommunal bindende plan for et afgrænset område |
| Byggelinje | Linje inden for hvilken der ikke må bygges |
| Planklagenævnet | Statslig klageinstans for bygge- og plansager |
| Injunktion | Civilt retskrav om at en adfærd ophører (forbud) |
| Hævd | Ret erhvervet ved langvarig besiddelse (20 år) |
| Fogedretten | Domstolsafdeling der gennemfører tvangsfuldbyrdelse |

## Instansoversigt

| Problem | Instans |
|---------|---------|
| Hegn og skel | Hegnsynet |
| Nabotræer | Hegnsynet |
| Støj, røg, lugt | Kommunens tekniske forvaltning |
| Ulovligt byggeri | Kommunens byggemyndighed |
| Byggetilladelse | Planklagenævnet (inden 4 uger) |
| Lokalplansbrud | Kommunen → Planklagenævnet |
| Servitutbrud | Domstolene |
| Skelstrid | Landinspektør → Hegnsynet → Domstolene |
